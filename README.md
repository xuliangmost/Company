# 机器人云平台项目

## 使用技术
`react reflux es6 less gulp webpack react-router ant脚手架 ,and.design组件相关`

## 安装方式
`执行 npm install`

## 运行方式
`执行 npm start`


## 打开方式
`浏览器打开(http://127.0.0.1:8080)(http://localhost:8080)`

## 文件结构

以下只对和项目相关的重要文件做了说明，

```
--proxy.config.js 本地测试接口
--webpack.config.js
--src 项目开发代码
----actions actions
----components 自定义组件
----entries 入口文件
----pages 逻辑页相关
----routes 路由
----store 前端数据存储 
----modules 项目依赖相关模块
------data-interface 数据接口模块
------global-config 全局配置
------static 静态资源文件
------tool 工具相关函数

----apply 会议申请模块
------check 会议审核模块
------invalid 作废会诊模块


------webpack setting
------https://webpack.js.org/guides/code-splitting-libraries/



```
